Objetos ---------------------------------
objeto literal é um objeto criado com a notação literal, ou seja: 
uma lista de chave e valores dentro de chaves { },
que atribuímos a uma variável para que o valor possa ser acessado depois.
formado por chave e valor
usados para armazenar lista de mais de um tipo de informação exemplo nome e idade
        const cliente = {
            nome: "maria",
            idade: 20,
            cpf:"12312312399",
            email:"maria@gmail.com"
        }
        //objeto: cliente , chave: nome com valor maria, chave idade com valor 20

é possivel acessar o valor da chave, alterar e deletar
console.log(cliente.nome); // acessa valor
cliente.nome = "Pedro" //se não existe adiciona, se existe atualiza valor
delete cliente.nome //remove

JSON ---------------
JavaScript Object Notation ou “notação de objeto JavaScript”
formato que utiliza a sintaxe de objetos e arrays do JavaScript,
forma mais comum de estrutura para transferência de dados entre cliente/servidor,
e tem sido utilizado mesmo em programas que não utilizam JavaScript
formato criado para transferência de dados, sendo assim é necessário convertê-lo para um objeto JavaScript para que os dados possam ser utilizados em um programa. Para isso, existem dois métodos nativos:

JSON.parse(): converte JSON para um objeto JavaScript;
JSON.stringify(): converte um objeto JavaScript para o formato JSON.

O processo de converter estruturas de dados em sequências de bytes ou caracteres
é chamado de serialização ou marshaling

array-------------------------------------
tem os elementos que são acessados pelo indice
usadas para armazenas lista de informaçoes do mesmo tipo por exemplo lista de nomes
array interpretado como objeto pelo javascript

metodo: substring() - puxa valores numericos que deseja do valor da chave
método Object.create() - cria um novo objeto a partir do objeto passado, uma copia
this - faz referencia ao proprio objeto
ForIn() - metodo de objetos for(let variavel in objeto){} para cada 
Object.keys() - cria um array com todas as cheves do objeto
Object.values() - cria um array com os valores do objeto
Object.entries() - cria um array para cada chave do objeto

operador 
... - “espalha” os elementos de um array (ou propriedades de um objeto) dentro de outro.
